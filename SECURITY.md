# Segurança do Seu Funcionário

Não publique vulnerabilidades que possam expor dados. Use o canal de suporte
configurado no aplicativo e informe a versão, o fluxo afetado e passos mínimos
para reproduzir. Não envie senhas, tokens, chaves de API ou dados reais de
clientes.

O projeto prioriza correção imediata para acesso entre contas, tomada de conta,
perda de dados, execução sem confirmação e exposição de segredos.

Consulte `docs/RESPOSTA_A_INCIDENTES_E_LGPD.md` para o procedimento interno.
